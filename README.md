# UART Log Plot
This little software plots and logs your data sent over UART.

Expected Format: `<group>/<signal>[unit]: <value>`

Example: `Temperature/SHT41 Temperature[Â°C]: 22.36`

Just run the python script `uartLogPlot.py` or the `/dist/uartLogPlot.exe`
